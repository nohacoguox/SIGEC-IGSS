Logo IGSS para el PDF SIAF-A-01
================================

Para que el PDF generado por el backend muestre el logo del IGSS, copia aquí el archivo del logo con el nombre:

  logo-igss.png

Puedes copiarlo desde:
  frontend/src/assets/images/logo-igss.png

(o desde frontend/public/images/ si lo tienes como logoIgss.png, renómbralo a logo-igss.png)

El PDF lo buscará en: backend/assets/logo-igss.png
